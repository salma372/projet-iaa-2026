# Projet Machine Learning - Classification d'images

## Contexte 
Projet réalisé en L3 dans le cadre de l'UE Introduction à l'Apprentissage Automatique

## Objectif 
L'objectif de ce projet est de évelopper un modèle de classification d'images capable de distinguer automatiquement deux classes :
-Mer (+1)
-Ailleurs(-1)

## Données
Les images utilisées pour l'apprentissage sont dans les sous dossiers Mer et Ailleurs du dossier Init. Puis dans un second temps les images du dossier dataCC2Test seront aussi utilisées pour l'apprentissage après les avoir utilisées pour tester l'efficacite du modèle
